TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Compilation Error:
Can't find symbol
└─ Check spelling and capitalization
└─ Ensure library imports are correct
└─ Make sure all classes are in same package

Runtime Error:
Exception in thread "main"
└─ Check error message carefully
└─ Look for line number in stack trace
└─ Review input values

Output Not Showing:
└─ Ensure System.out.println() is used
└─ Check for infinite loops
└─ Verify program logic

File Not Found:
└─ Check file exists in correct directory
